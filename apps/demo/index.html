Headings and Structure

Typography is the art and technique of arranging type to make written language legible, readable, and appealing when displayed. It involves selecting typefaces, point sizes, line lengths, line-spacing, and letter-spacing.

History and Evolution

The arrangement of type involves selecting typefaces, point sizes, line lengths, line-spacing (leading), and letter-spacing (tracking), as well as adjusting the space between pairs of letters (kerning).

Modern Digital Typography

Today's typography extends far beyond traditional print media into digital interfaces, responsive design, and interactive experiences.

Kerning
The process of adjusting the spacing between characters in a proportional font.
Leading
The space between lines of text, also known as line spacing.

Quotations and Citations

Typography is the craft of endowing human language with a durable visual form, and thus with an independent existence.

— Robert Bringhurst, The Elements of Typographic Style

As The Elements of Typographic Style emphasizes, good typography is invisible to the reader.

Code and Technical Content

When discussing code, we often need to show both inline code like font-family: 'Inter', sans-serif; and code blocks: 

/* CSS Typography Example */
.prose {
  font-family: 'Inter', system-ui, sans-serif;
  font-size: 1rem;
  line-height: 1.75;
  color: #374151;
  max-width: 65ch;
}

.prose h1 {
  font-size: 2.25em;
  font-weight: 800;
  line-height: 1.1111;
}

For keyboard shortcuts, use the Cmd + C notation to copy text.
